# AI Vibe-Coding Resources

## Knowledge

- [OpenAI Codex docs](https://developers.openai.com/codex)
  Codex nima qila olishini tushunish uchun asosiy rasmiy manba. Use for: agentga task berish, codebase tushuntirish, bug fix va feature workflows.
- [Codex web guide](https://developers.openai.com/codex/cloud)
  Codex cloud background tasklar, parallel ish, va repo ichida agent bilan ishlash haqida. Use for: qachon local agent, qachon cloud agent ishlatishni tanlash.
- [OpenAI: Introducing Codex](https://openai.com/index/introducing-codex/)
  Codex konsepti: software engineering agent task bajaradi, kod o'qiydi, test yuritadi, PR taklif qiladi. Use for: mental model.
- [GitHub Copilot best practices](https://docs.github.com/en/copilot/get-started/best-practices)
  AI coding assistant bilan ishlaganda prompt, tekshirish, va agent ishini nazorat qilish bo'yicha amaliy tavsiyalar. Use for: "AI yozdi, endi men nima qilaman?" savoli.
- [GitHub: Prompt engineering for Copilot Chat](https://docs.github.com/copilot/concepts/prompt-engineering-for-copilot-chat)
  Promptni aniq, kontekstli, va tekshiriladigan qilish haqida. Use for: yaxshi topshiriq yozishni mashq qilish.
- [Atlassian: Acceptance criteria](https://www.atlassian.com/work-management/project-management/acceptance-criteria)
  Ish tugaganini qanday bilamiz degan savolga javob. Use for: har bir AI task uchun "done" shartlarini yozish.
- [Atlassian: User stories](https://www.atlassian.com/agile/project-management/user-stories)
  G'oyani user nuqtai nazaridan kichik, test qilinadigan storyga aylantirish. Use for: product fikrni agentga tushunarli formatga solish.

## Wisdom (Communities)

- [OpenAI Developer Community](https://community.openai.com/)
  Codex va OpenAI tooling bo'yicha real foydalanuvchi tajribalari. Use for: agent workflow, prompt, va yangi imkoniyatlar haqida real misollar.
- [GitHub Community: Copilot](https://github.com/orgs/community/discussions/categories/copilot)
  Copilot va AI-assisted development bo'yicha savol-javoblar. Use for: umumiy AI coding muammolarini solishtirish.
